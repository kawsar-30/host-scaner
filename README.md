# Host Scanner Tool

--------------------------------------------------------------------------------------

### 📌 Description
**Host Scanner** is a professional Python-based network reconnaissance project tool. It integrates **Nmap** capabilities with Python automation to identify open ports, services, operating systems, and MAC addresses on target hosts.

It is designed to demonstrate:
* **Network Enumeration:** Identifying active services and versions.
* **OS Fingerprinting:** Detecting target machine Operating Systems.
* **Data Automation:** Exporting scan results into TXT, CSV, and JSON formats.

---

### 🚀 Usage Instructions

#### **Command Format**
The tool follows a straightforward command structure:

```bash
python3 host_scaning.py <port_number> <ip_address>
```


---
### 📂 Output Reports
After every scan, the tool automatically generates:
* **host_scaning_report.txt** (Text Summary)
* **host_scaning_report.csv** (Spreadsheet Data)
* **host_scaning_report.json** (Structured JSON)
